## How to run

### Development Environment

**현재는 개발 환경만 구현**하였습니다.  
다음 명령어를 터미널에 입력해주세요.

```
npm run dev:webpack
```

이후, `http://localhost:8080/` 에서 결과를 확인하면 됩니다 :)
